Week 2

Capstone 1 - Build and E-commerce Website

What We Learned

We mainly focused on making a responsive E-commerce website using HTML and CSS but we also learned to use bootstap and apply the features to our websites to add to the users experience. As well as how to use custom media query to change the way our pages looked on different devices.

Others:
  • How to make a Github Repository
  • How to make an appropriate directory structure and the file naming convetion
  • Consistent look and feel throughout the site
  • How to make comments on CSS and HTML
  • How to check for errors and correct them through validators
  • How to use the inspect option in our browsers to facilitate our projects
  • How to commit using VS code and the command line/Git Bash
Challenges

This Capstone made us challenge our knowledge and ability to apply HTML, CSS, and Bootstrap in a responsive manner. As well as learning to make a proper repository.

Others:
  • Creating a ReadMe and adding images to the file
  • Using Features in Bootstrap that we didn't practice
  • Using new CSS and HTML features from W3schools and applying them properly
  • Using the custom media query to make our pages fit properly in a desktop and mobile device
Key Vocabulary
  1. Git Branches: a branch is a parallel line of development that allows multiple changes to be worked on simultaneously without affecting the main or master code.
  2. Console Tab: a command line interpreter through which you can quickly enable a script engine and enter commands to act on the script.
  3. Semantic HTML Elements: elements that clearly describe their meaning in a human and machine readable way. ex: the header tag
  4. Responsive Behaviour: when the websites are able to adapt and respond to the user's behavior and environment.
  5. Directory: a file system cataloging structure which contains references to other computer files, and possibly other directories.
  6. Repository: the .git/ folder inside a project. This repository tracks all changes made to files in your project, building a history over time.
  7. Media Query: a key part of responsive web design, as they allow you to create different layouts depending on the size of the viewport, but they can also be used to detect other things about the environment your site is running on.
  8. BootStrap: the most popular CSS Framework for developing responsive and mobile-first websites.
  9. HTML Validation Attributes: allows you to validate the value of any HTML attribute of an element (e.g. href, src, alt, title, etc.).
