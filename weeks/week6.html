BOM and DOM

The Browser Object Model (BOM) encompasses all the objects provided by a web browser other than the actual document being displayed. It includes objects like window, navigator, screen, history, and location, allowing interaction with the browser environment. BOM facilitates tasks such as controlling browser behavior, managing windows and frames, handling navigation, and retrieving information about the user's environment and device.

The Document Object Model (DOM) is an interface that represents the structure of a web page as a tree-like structure where each node represents an element. It allows JavaScript and other scripting languages to manipulate the content, structure, and style of web documents dynamically. Through the DOM, developers can access, modify, and update elements and their attributes, enabling interactive and dynamic web page behavior.

getElementById

getElementById is a JavaScript method used to retrieve an HTML element from the document by specifying its unique identifier (ID). In HTML, you can assign a unique ID to an element using the id attribute. The getElementById method then allows you to access and manipulate that specific element in the Document Object Model (DOM).

Query Selectors

Query selectors in JavaScript are utilized to access and manipulate HTML elements dynamically within a webpage. Developers use these selectors by employing methods like querySelector() or querySelectorAll() with CSS-like syntax to target specific elements based on their IDs, classes, attributes, or element types, enabling efficient modification or retrieval of content on the page. These selectors provide a powerful means to interact with and modify the Document Object Model (DOM) of a web page, facilitating dynamic and responsive web development.
