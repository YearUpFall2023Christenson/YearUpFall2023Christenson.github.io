Capstone 3: Microblog Network

On week 12, we will be building a a microblogging website that shares short insights with other users.

Page descriptions

Landing/Login page

Introduce the app to the user by displaying the website name and a short description of the purpose or functionality of the app
Have a login feature that is functional.

Registration page

sign up form that will perform a fetch request and redirect the user back to the landing/login page

Posts page

User can access the this page by creating an account. They can view posts created by other users

Profile page

user can create posts that will be sent to the main feed in the Posts page

Skillsets needed

Understanding of Git commands, branching, merging, pull requests
Understanding of RESTful API concepts, HTTP methods (GET, POST, PUT, DELETE).
Effective communication, collaboration, and teamwork.
