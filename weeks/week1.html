Command Line Basics

New Concepts Introduced:

During this week, we were introduced to fundamental concepts of command line navigation, file manipulation, and basic commands.

Key Vocabulary Words and Definitions:

  • Command Line: The text-based interface used to interact with the operating system.
  • Directory: A folder or container that holds files and other directories.
  • Command: An instruction given to the computer to perform a specific task.

Challenging Aspects:

Understanding the syntax of command line commands and navigating through directories efficiently posed initial challenges for many learners.

Workshops, Capstones, or Mini-Projects:

Throughout the week, students engaged in hands-on exercises to practice command line navigation, create directories, and manipulate files. Some of the more challegnging projects involved real-world scenarios requiring the use of problem-solving skills.
