Module 1

Key Concepts

  • Functions

    • A block of code that is designed to perform a specific task when called.
    A JavaScript function is defined by using:

    • Keyword: function
    • name of function
    • set of parentheses ( )
    • set of curly brackets { } containing body of function

  • Parameters

    • Values (arguments) that are passed through the functions in parentheses.

  • Variables

    • Using var, let, and const

  • Scoping

    • Scope is a word that us used to describe how long a variable lives and where it can be accessed.
