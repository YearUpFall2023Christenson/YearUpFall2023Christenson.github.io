Week 10

A brief summary of our week 10

  1. Object-Oriented Programming ( OOP )

    2. OOP is a way of writing code that designs around the data in a system

                Classes: are user-defined data types that act as the blueprint for individual objects.
            Properties: are data members that help identify the state of an object.
            Objects: are instances of a class created with specifically defined data, or properties.
            Methods: are functions that are defined in a class and operate on the data in an object.
  2. ES6 classes

    3. ES 6 gives us a new syntax to create objects. Rather than use classes to create the objects.

  3. Constructor

    4. Create a constructor that will initialize the properties of an objects created by the class (Use the keyword this to reference the properties)

  4. Creating Object using a class

    5. To create an object we use new keyword followed by the name of the class.

  5. Defining Methods

    6. Methods are functions that are wrapped up with object and operate upon the data in the object.

  6. Enhancing the class
  7. Calling One Method from Another
  8. Extending a class
